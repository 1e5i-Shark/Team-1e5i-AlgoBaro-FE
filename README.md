## 🔎 프로젝트 소개

**알고바로 (AlgoBaro)**
바로바로 함께 푸는 알고리즘 테스트

실제 코딩테스트 유사하게 시간 제한과 기본 레퍼런스만 제공되는 환경에서 코딩테스트를 다 같이 풀고, 여러 사람의 풀이 코드를 공유할 수 있는 서비스
